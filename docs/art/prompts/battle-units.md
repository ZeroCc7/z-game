# 战斗单位提示词

统一要求：

- 原创角色，不使用现有游戏 IP。
- 正式战斗单位不要只交付单张平面图。每个角色、宠物、敌人至少需要同一身份的 2x2 四方向多角度表：朝左上、朝右上、朝左下、朝右下。
- 战斗场景优先使用我方朝左上、敌方朝右下。
- 每个方向必须保持同一服装、轮廓、配色、武器/尾巴/角等身份标记，体型和脚底锚点一致。
- 3/4 顶视战斗视角。
- 透明背景 PNG；如果先生成原始表，使用纯 #FF00FF 背景再做抠图。
- 适合放在 1024x768 回合制战斗场景里。
- 不要文字、UI、血条、光标或伤害数字。
- 建议交付：每个单位一张 2x2 四方向表和一个运行时单帧导出；单帧只作为当前原型临时资产。
